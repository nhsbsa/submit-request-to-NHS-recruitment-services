Submit a request to NHS Recruitment Services

Use this portal to request changes to the HR records of your employees. You’ll be able add documents to support the change.

Some changes can be made in the Manage Self-Service (MSS) (opens in a new tab). You should attempt to make these changes in the MSS before submitting a request using this portal.

Changes you can make through this portal:

  • all allowances
  • annual leave amendments
  • band or grade
  • career break
  • career progression
  • extension of probation
  • flexible working
  • future date changes
  • maternity, paternity, shared parental or adoption leave details
  • new starter form
  • pay step points
  • personal salaries
  • promotions
  • removal of absence and sickness
  • term-time working
  • unpaid leave

HR Shared Services will retain responsibility for contractors and flexi-retirement.

Before you start

You'll be asked to fill out details about the employee. You can find employee employment details in the Electronic Staff Record (ESR).
